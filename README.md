# BreadMC - Premium Minecraft Server Website

Trải nghiệm website máy chủ Minecraft chất lượng cao với thiết kế hiện đại, hiệu ứng mượt và tối ưu hóa hiệu suất.

## Công nghệ sử dụng

- **React 19** - Thư viện UI
- **Vite** - Build tool siêu nhanh
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first CSS
- **Framer Motion** - Animation
- **Lucide React** - Icons

## Tính năng chính

- ✅ Hero section ấn tượng với hiệu ứng parallax
- ✅ Hiệu ứng hạt từ trường cao cấp
- ✅ Thư viện hình ảnh (placeholder)
- ✅ FAQ với accordion
- ✅ Cộng đồng mạng xã hội
- ✅ Easter egg ẩn (Konami code)
- ✅ Responsive hoàn chỉnh
- ✅ Tối ưu hóa production

## Cài đặt và chạy

### Cài đặt dependencies

```bash
npm install
```

### Chạy development server

```bash
npm run dev
```

### Build production

```bash
npm run build
```

### Xem trước production build

```bash
npm run preview
```
